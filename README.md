# Win系统快捷键扩展


## 项目效果
- 修改键盘快捷键
    - 通过键盘驱动
    - 通过Ahk
- 拓展额外按键
    - 快捷键替换（Ctrl+Alt+Shift -> RWin)
    - （F13-F24）
    - 全局快捷键
- 自定义文本替换 /time/ -> {current_time}
- 屏蔽部分系统默认的快捷键
- 系统默认快捷键功能替换
- 模拟鼠标操作
- 窗口快捷操作
    - 窗口位置、大小
- 通过快捷键快速启动应用
- 软件中自定义快捷键
    - 宏脚本
    - 快捷键修改
- 软件中展示快捷键帮助
- 软件弹窗自动居中

|-------------------|------------------------|--------------------------------|
